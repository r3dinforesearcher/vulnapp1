Git Basics

git init
Initializes a new Git repository in the current folder.
git add .
Stages all changes for commit.
git commit -m "Initial commit"
Commits staged files with a message.
git remote add origin https://github.com/username/repo.git
Links local repository to remote GitHub repo.
git push -u origin main
Pushes commits to GitHub on the main branch.

Docker Basics

docker build -t myapp .
Builds Docker image using Dockerfile in current directory.
docker run -d -p 8080:80 myapp
Runs container in background and maps port 8080 on host to 80 on container.

Docker Networking

docker network create mynetwork
Creates a custom Docker network.
docker run -d --name web --network mynetwork nginx
Runs a web server container attached to custom network.

Docker Security

docker scan myapp
Scans the Docker image for vulnerabilities.
docker run --read-only --cap-drop=ALL myapp
Runs container in read-only mode with no additional privileges.

Deployment

docker tag myapp username/myapp:latest
Tags Docker image for Docker Hub.
docker push username/myapp:latest
Pushes tagged image to Docker Hub.

Project 1: Static Website with Git + Docker

  • Create `index.html` with a simple welcome message.
  • Create a `Dockerfile`:
    FROM nginx:alpine
    COPY . /usr/share/nginx/html
  • Build the image and run it:
    docker build -t static-web .
    docker run -d -p 8080:80 static-web
  • Push to GitHub using Git commands shown earlier.

Project 2: Node.js App with Git + Docker + Compose

  • Create a Node.js `app.js`:
    • const http = require('http');
    http.createServer((req, res) => {
      res.end("Hello from Node.js!");
    }).listen(3000);
  • Create `Dockerfile`:
    • FROM node:alpine
    COPY . .
    CMD ["node", "app.js"]
  • Create `docker-compose.yml`:
    • version: '3'
    services:
      web:
        build: .
        ports:
          - "3000:3000"
  • Build and run:
    docker-compose up --build
  • Access app at http://localhost:3000
